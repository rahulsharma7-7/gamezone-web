📌 Project Title
GameZone – Online Game Store Website   https://gamezone-web.netlify.app/

📖 Description
GameZone is a simple and modern web-based project developed using HTML, CSS, and JavaScript. This website allows users to explore, buy, and download popular games through an interactive and user-friendly interface.

The main goal of this project is to demonstrate basic front-end development skills, including layout design, styling, and adding interactivity using JavaScript. It is designed especially for beginners and academic project submissions.

🚀 Features
🎮 Browse popular games

💰 View game prices

🛒 Buy game option (demo alert system)

⬇ Download game option (demo functionality)

🎨 Clean and modern UI design

✨ Smooth hover effects and animations

📱 Beginner-friendly structure

🛠 Technologies Used
HTML5 – Structure of the website

CSS3 – Styling and layout design

JavaScript – Interactivity and functionality

▶ How to Run
Download or clone the repository

Open the project folder

Double click on index.html

Run in any web browser

This project is created as a BCA First Year mini project to understand the fundamentals of web development and how different technologies work together to build a functional website.
